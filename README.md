#SwiftUICentered

## Installation

To install this package, import 'https://github.com/cb-raghavendra/SwiftUICentered.git' in Swift Package Manager

## Usage example 

```Swift

struct ContentView : View {
  var body : some View {
    VStack {
      Text("Centered")
      .centeredHorizontally()
    }
  }
}

```
